API Title: Movie Features

A server-side web application that displays movie titles, genres, and directors. The user will be able to sign up, update their personal information, and create a list of their favorite movies.

Endpoint URL HTTP Method Endpoint Description Query Parameters Request body data format w/ example Response body data format
/movies GET Get a list of all movies request + response None A JSON object indluding the titles of all movies
/movies/:movieTitle GET Return features about movie by title request + response None A JSON object holding data about all movie details
/genre/:genreName GET Return data about the movie by genre request + response None A JSON object holding data about a movie's genre
/director/:directorName GET Return data about a movie director request + response None A JSON object holding data about the movie's director
/users POST Create a New User request + response A JSON objust with new user: Juliet O'hara A JSON object hold the data about a user {name: "Juliet O'hara", username: detective123}
/users/:Username PUT Update User's Information request + response None A JSON object holds the updated username for the user {name: "Juliet O'hara", username: psyschoffice}
/users/:Username/movies/:movieId POST Add movie to Favorites List request + response Movie Added to Favorites {movieId: "64507fc2ed070ba0594ec4d0"} A message that movie by its objectId has been successfully added to Favorite's List
/users/:Username/movies/:movieId DELETE Allow user to delete a movie from Favorites List request + response None A message that movie has been removed from Favorite's List
users/:Username DELETE Allow existing user to deregister account request + response None A message that user has been successfully deleted
